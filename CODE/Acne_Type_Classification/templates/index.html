Model Core Architecture

Our system implements an advanced, hybrid deep learning pipeline tailored specifically for precise multi-ethnic acne severity scoring and spot localization. By combining real-time detectors with state-of-the-art transfer learning extractors, DERMA.AI delivers medical-grade precision.

YOLOv5 Spot Localization

Employs a modified **YOLOv5** architecture for single-stage, real-time bounding box prediction, localizing individual lesions (comedones, papules, pustules, nodules) with extreme spatial confidence.

Hybrid Transfer Learning

Integrates pre-trained **VGG16** and **ResNet50** backbones configured with customized global pooling and dense heads to extract premium high-dimensional dermatological features.

GAN-Based Minority Augmentation

Employs **DCGANs** to generate synthetic, high-fidelity severe acne samples, successfully mitigating class imbalance and enhancing model generalization across minor classes.
